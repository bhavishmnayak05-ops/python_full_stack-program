# Topic 2: Python Syntax, Variables, and Data Types

## 1. Basic Syntax
- Python uses **indentation** (spaces) to define blocks of code.
- **Comments** start with a `#` symbol. They are ignored by Python.

## 2. Variables
Variables are like boxes that store data. You don't need to declare their type.
```python
name = "Sriram"  # String
age = 25         # Integer
```

## 3. Basic Data Types
- **int:** Whole numbers (e.g., `10`, `-5`)
- **float:** Numbers with decimals (e.g., `10.5`, `3.14`)
- **str (String):** Text wrapped in quotes (e.g., `"Hello"`)
- **bool (Boolean):** Either `True` or `False`.

## 4. Checking Data Type
You can use the `type()` function to check the type of a variable.
```python
print(type(age))  # Output: <class 'int'>
```
