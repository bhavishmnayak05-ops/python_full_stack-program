# Topic 5: Data Structures

Python has four built-in data structures used to store collections of data.

## 1. List
- **Ordered** and **Changeable**.
- Allows duplicate members.
- Syntax: `my_list = [1, 2, 3]`

## 2. Tuple
- **Ordered** but **Unchangeable** (Immutable).
- Allows duplicate members.
- Syntax: `my_tuple = (1, 2, 3)`

## 3. Set
- **Unordered** and **Unindexed**.
- No duplicate members.
- Syntax: `my_set = {1, 2, 3}`

## 4. Dictionary
- **Ordered** (as of Python 3.7) and **Changeable**.
- Stores data in **Key:Value** pairs.
- No duplicate keys.
- Syntax: `my_dict = {"name": "Sriram", "age": 25}`
