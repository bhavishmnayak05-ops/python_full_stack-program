# Topic 6: String Handling

Strings are sequences of characters. Python provides many built-in methods to manipulate strings.

## 1. Creating Strings
```python
s1 = 'Single quotes'
s2 = "Double quotes"
s3 = """Triple quotes for
multi-line strings"""
```

## 2. Indexing and Slicing
Each character has an index (starting from 0).
- `s[0]` - First character
- `s[-1]` - Last character
- `s[0:5]` - First 5 characters (index 0 to 4)

## 3. Common Methods
- `upper()`: Convert to uppercase.
- `lower()`: Convert to lowercase.
- `replace(old, new)`: Replace text.
- `split()`: Split string into a list.
- `strip()`: Remove leading/trailing spaces.
