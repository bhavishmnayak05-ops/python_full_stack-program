# Topic 1: Introduction to Python & Environment Setup

## What is Python?
Python is a simple, easy-to-learn programming language. It is very popular for web development, data science, and automation.

### Key Features:
- **Easy to Read:** Python code looks like simple English.
- **Large Community:** Many people use it, so it's easy to find help.
- **Versatile:** You can use it for almost anything!

## Environment Setup

### 1. Install Python
- Go to [python.org](https://www.python.org/) and download the latest version for your OS.
- During installation, make sure to check **"Add Python to PATH"**.

### 2. Install VS Code
- Download Visual Studio Code from [code.visualstudio.com](https://code.visualstudio.com/).

### 3. Install Python Extension in VS Code
- Open VS Code.
- Click on the Extensions icon on the left.
- Search for "Python" and install the one by Microsoft.

## Your First Step
To check if Python is installed correctly, open your terminal and type:
```bash
python --version
```
If you see a version number (like `Python 3.10.x`), you are ready to go!
